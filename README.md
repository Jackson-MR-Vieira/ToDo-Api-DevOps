# 🚀 ToDo API DevOps (FastAPI)

Este é um projeto simples de API de tarefas desenvolvido com FastAPI, criado com o objetivo de praticar conceitos de DevOps.

---

## 📌 Sobre o projeto

A API permite:

* Criar tarefas
* Listar tarefas

Este projeto foi desenvolvido como parte de uma atividade de DevOps, utilizando GitHub para versionamento e controle de código.

---

## 🛠️ Tecnologias utilizadas

* Python 3
* FastAPI
* Uvicorn
* Pytest

---

## ▶️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/Jackson-MR-Vieira/simple-todo-api-fastapi.git
cd simple-todo-api-fastapi
```

### 2. Criar ambiente virtual (opcional, mas recomendado)

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

### 3. Instalar dependências

```bash
pip install -r requirements.txt
```

### 4. Executar a API

```bash
uvicorn main:app --reload
```

---

## 🧪 Executar testes

```bash
pytest
```
